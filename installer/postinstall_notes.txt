Supermix Qwen Desktop installs a lightweight launcher executable.

Before first launch:
- Make sure Python is installed and available on PATH as "python".
- Keep the local Qwen2.5-0.5B-Instruct base model snapshot available in the Hugging Face cache.
- The packaged adapter is already bundled with this installer.

When you open the app, it starts the local chat server automatically and shows the chat UI in its own desktop window.
